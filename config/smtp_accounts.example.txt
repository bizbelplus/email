# Формат строки:
# host|port|username|password|from_email|from_name|use_tls|use_ssl|timeout_seconds
#
# Расшифровка полей:
# host             — SMTP-сервер, например smtp.gmail.com
# port             — SMTP-порт, обычно 587 для TLS или 465 для SSL
# username         — логин SMTP-аккаунта
# password         — пароль или app password SMTP-аккаунта
# from_email       — email отправителя, который увидит получатель
# from_name        — имя отправителя, которое увидит получатель
# use_tls          — использовать TLS: true/false
# use_ssl          — использовать SSL: true/false
# timeout_seconds  — таймаут SMTP-соединения в секундах
#
# Важно:
# - use_tls=true и use_ssl=true одновременно включать нельзя
# - если use_tls/use_ssl/timeout_seconds не указаны, используются значения по умолчанию:
#   use_tls=true, use_ssl=false, timeout_seconds=30
# - одна строка = один SMTP-аккаунт
# - приложение использует эти строки для ротации/параллельной отправки
# Поля use_tls, use_ssl и timeout_seconds опциональны.
# Пустые строки и строки, начинающиеся с #, игнорируются.
